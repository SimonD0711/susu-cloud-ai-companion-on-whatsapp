# Susu Cloud - AI Companion on WhatsApp

An AI-powered WhatsApp companion, now backed by the Brain Bridge architecture.

## Architecture
- **Brain Bridge**: Robust bridge-backed brain architecture (susu_brain_bridge.py).
- **Memory System**: Multi-tier memory with bucket cascade decay.
- **Task Intelligence**: Task-type classification (education, Q&A, emotional support).

## Setup
- Environment managed via `/etc/wa-agent.env`.
- Services managed via systemd.
